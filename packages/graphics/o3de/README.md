###  O3DE Docker Setup

This folder contains the necessary files to build and run a Docker container with O3DE installed using Ryzers.

### Build the Docker Image

```sh
ryzers build o3de
ryzers run
```

### Launch o3de
```sh
o3de
```

The menu will display, so that you can create and build a new project.
